### Project Context
- **Format**: `{user_name}-{project_name}`
- **Example**: `test_user-testproject1`
- **Separators**: Use hyphen (`-`) to separate user and project names
- **Valid Examples**:
  - `test_user-myproject`
  - `john-demosite`
  - `alice-webapp`
